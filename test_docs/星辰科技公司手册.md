# 星辰科技公司员工手册

## 公司简介

星辰科技有限公司成立于 2018 年 3 月 15 日，总部位于上海市浦东新区张江高科技园区。公司创始人兼 CEO 是李明华博士，他毕业于清华大学计算机系，曾在谷歌工作 8 年。

公司目前拥有员工 356 人，其中研发人员占比 65%。2024 年公司营收达到 2.8 亿元人民币，同比增长 42%。

## 产品线

### 1. 星云数据库（StarCloud DB）

星云数据库是公司的核心产品，于 2019 年 6 月首次发布。它是一款分布式数据库，支持每秒处理 100 万次查询（QPS），最大支持 500 个节点的集群部署。

**定价方案**：
- 社区版：免费
- 专业版：每节点每月 2,000 元
- 企业版：每节点每月 5,000 元（含 7×24 技术支持）

### 2. 星盾安全套件（StarShield）

星盾是公司于 2021 年推出的网络安全产品。主要功能包括：
- 入侵检测系统（IDS）
- 防火墙管理
- 漏洞扫描
- 日志审计

星盾目前服务超过 200 家企业客户，包括 3 家世界 500 强公司。

### 3. 星链 API 网关（StarLink）

星链于 2023 年 1 月上线，是一款高性能 API 网关。核心特性：
- 支持每秒 50 万请求
- 延迟低于 5 毫秒
- 内置限流、熔断、降级功能
- 支持 gRPC 和 REST 协议

## 员工福利

### 薪酬体系

- 每年 13 薪 + 年终奖金（0-6 个月工资）
- 股票期权：工作满 2 年可获得
- 年度调薪：每年 4 月进行，平均涨幅 8-15%

### 假期制度

| 工作年限 | 年假天数 |
|----------|----------|
| 1 年以下 | 5 天 |
| 1-3 年 | 10 天 |
| 3-5 年 | 15 天 |
| 5 年以上 | 20 天 |

另外，公司提供：
- 带薪病假：每年 12 天
- 婚假：15 天
- 产假：女员工 180 天，男员工陪产假 20 天

### 其他福利

1. **餐饮补贴**：每月 600 元
2. **交通补贴**：每月 500 元
3. **健身房**：公司自建健身房，免费使用
4. **年度体检**：每年一次全面体检，价值 2000 元
5. **团建活动**：每季度一次，预算每人 500 元

## 技术栈

公司主要使用以下技术：

**后端**：
- 语言：Go（主力）、Rust、Python
- 框架：Gin、Actix-web
- 数据库：PostgreSQL、Redis、自研星云数据库

**前端**：
- 框架：React 18 + TypeScript
- 状态管理：Zustand
- UI 组件：Ant Design 5.0

**基础设施**：
- 容器化：Kubernetes + Docker
- CI/CD：GitLab CI
- 云服务商：阿里云（主）、AWS（海外）
- 监控：Prometheus + Grafana
