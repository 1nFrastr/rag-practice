## 💡 调优建议

如果效果不好，可以尝试：

1. **调整分片大小**：`config.py` 中的 `CHUNK_SIZE`
   - 太小：上下文不够
   - 太大：检索不精确

2. **调整重叠**：`CHUNK_OVERLAP`
   - 增加重叠可改善跨分片信息的检索

3. **增加返回的文档数量**：在 query_engine 中调整 `similarity_top_k`

4. **使用更好的模型**：
   - 嵌入模型：text-embedding-3-large
   - LLM：gpt-4o 或 claude-3
