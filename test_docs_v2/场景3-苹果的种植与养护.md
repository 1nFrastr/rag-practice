# 苹果的种植与养护指南

## 品种选择

常见的苹果品种：
- **红富士**：口感脆甜，最受欢迎
- **金冠**：果肉细腻，适合鲜食
- **嘎拉**：早熟品种，8月即可采摘
- **红星**：酸甜适中，适合做果汁

## 种植条件

### 气候要求
- 年均温度：8-14℃
- 冬季需要一定低温时长（冷凉需求）
- 降雨量：500-800mm/年

### 土壤要求
- pH值：6.0-7.0
- 排水良好的沙壤土
- 有机质含量2%以上

## 年度管理日历

| 月份 | 主要工作 |
|------|----------|
| 1-2月 | 冬季修剪 |
| 3-4月 | 施基肥、防治病虫害 |
| 5-6月 | 疏花疏果、套袋 |
| 7-8月 | 追肥、灌溉 |
| 9-10月 | 采收、储存 |
| 11-12月 | 清园、防冻 |

## 常见问题

### 苹果树不结果？
可能原因：树龄不够（需5年以上）、缺乏授粉树、营养过剩导致徒长

### 果实有病斑？
可能是轮纹病或炭疽病，需及时喷洒杀菌剂
