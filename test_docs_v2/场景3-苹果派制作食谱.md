# 经典苹果派制作食谱

## 食材准备

### 派皮材料
- 低筋面粉：250g
- 黄油：125g（冷藏）
- 糖粉：50g
- 蛋黄：1个
- 冰水：30ml

### 苹果馅料
- 苹果：4个（约800g，推荐红富士）
- 细砂糖：80g
- 肉桂粉：1茶匙
- 柠檬汁：1汤匙
- 玉米淀粉：2汤匙

## 制作步骤

### 1. 制作派皮（提前1天）
1. 面粉过筛，加入糖粉
2. 冷黄油切小块，加入面粉中
3. 用手指搓成粗砂状
4. 加入蛋黄和冰水，揉成团
5. 用保鲜膜包好，冷藏过夜

### 2. 准备苹果馅
1. 苹果去皮去核，切薄片
2. 加入糖、肉桂粉、柠檬汁拌匀
3. 撒上玉米淀粉拌匀

### 3. 组装烘烤
1. 派皮擀成3mm厚，铺入派盘
2. 填入苹果馅
3. 表面铺上网格派皮
4. 刷蛋液
5. 预热烤箱180℃，烤45-50分钟

## 小贴士

- 派皮要保持低温，防止黄油融化
- 苹果选择偏酸的品种口感更好
- 出炉后放凉再切，馅料更成型
