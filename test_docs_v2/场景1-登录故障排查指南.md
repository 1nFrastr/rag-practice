# 登录故障排查指南

当用户反馈登录失败时，请按以下步骤排查：

## 常见登录失败原因及解决方法

### 1. 密码错误
- 检查大小写是否正确
- 尝试重置密码：点击"忘记密码"链接

### 2. 账号被锁定
- 连续输错5次密码会触发锁定
- 解决方法：等待30分钟后自动解锁，或联系管理员手动解锁

### 3. 验证码失效
- 验证码有效期为60秒
- 解决方法：点击刷新按钮获取新验证码

### 4. 网络超时
- 检查网络连接状态
- 尝试切换网络（如从WiFi切换到4G）

### 5. 浏览器兼容性
- 清除浏览器缓存和Cookie
- 尝试使用Chrome或Firefox浏览器

如果以上方法都无法解决，请联系技术支持：support@example.com
